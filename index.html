Introduction à la programmation Python

Qu'est-ce que Python?

Python est un langage de programmation de haut niveau et interprété, connu pour sa syntaxe facile à lire. Il est largement utilisé dans le développement web, l'analyse de données, l'intelligence artificielle, et plus encore.

Types de données de base

entier = 10  # Un nombre entier
flottant = 10.5  # Un nombre à virgule flottante
chaine = "Bonjour"  # Une chaîne de caractères
booleen = True  # Vrai ou Faux (True ou False)

Entrée et Sortie

# Entrée de l'utilisateur
nom = input("Comment vous appelez-vous ? ")

# Sortie vers la console
print("Bonjour, " + nom)

Variables

age = 25  # Déclare une variable nommée age qui contient la valeur 25

Conditions

if age > 18:
    print("Vous êtes majeur.")
else:
    print("Vous êtes mineur.")

Listes

fruits = ["pomme", "banane", "cerise"]
print(fruits[0])  # Affiche 'pomme'

Boucles

# Boucle for
for fruit in fruits:
    print(fruit)

# Boucle while
i = 0
while i < len(fruits):
    print(fruits[i])
    i += 1

Fonctions

def saluer(personne):
    print("Bonjour, " + personne)

saluer("Alice")  # Appelle la fonction saluer avec 'Alice' comme argument

Opérateurs

# Opérateurs arithmétiques
a = 10
b = 3
print(a + b)  # Addition
print(a - b)  # Soustraction
print(a * b)  # Multiplication
print(a / b)  # Division
print(a % b)  # Modulo
print(a ** b) # Exponentiation

# Opérateurs de comparaison
print(a == b) # Égal à
print(a != b) # Différent de
print(a < b)  # Inférieur à
print(a > b)  # Supérieur à
print(a <= b) # Inférieur ou égal à
print(a >= b) # Supérieur ou égal à

# Opérateurs logiques
print(a > 5 and b < 5) # ET logique
print(a > 5 or b < 5)  # OU logique
print(not(a > 5))      # NON logique

Divers

# List Comprehensions
nombres = [1, 2, 3, 4, 5]
carrés = [x ** 2 for x in nombres]
print(carrés)

# Multiples affectations
x, y, z = 1, 2, 3

# Échanger des valeurs
x, y = y, x

# Utilisation de _ pour ignorer des valeurs
x, _, z = (1, 'ignoré', 3)

# Utiliser `enumerate` pour obtenir l'index et la valeur dans une boucle
for index, valeur in enumerate(nombres):
    print(index, valeur)
